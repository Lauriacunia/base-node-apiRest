# Proyecto Base para Node Js Api Rest

### Crear .env y completar variables:

- MONGO_USER
- MONGO_PASS
- DB_NAME

### Incluye dependencias

```
 "dependencies": {
    "cors": "^2.8.5",
    "dotenv": "^16.0.2",
    "express": "^4.18.1",
    "mongoose": "^6.6.1"
  }
```

### Correr con:

```
> npm star
```
